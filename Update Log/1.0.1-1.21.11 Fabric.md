# Subtitle Highlight Unofficial Reset - 更新日志

## 版本: 1.0.1-1.21.11 Fabric

### 更新日期: 2026-03-29

### 新增功能

1. **命令系统**
   - 实现了客户端命令 `/shur`
   - 支持配置重新加载、保存和各项设置的查询与修改
   - 命令输出支持多语言本地化

2. **模组 API**
   - 创建了 `com.qituo.shur.api` 包
   - 提供了 `SubtitleAPI` 类和 `SubtitleProcessor` 接口
   - 允许其他模组注册自定义字幕处理器，扩展字幕功能

3. **性能优化**
   - 实现了视锥体检查，优化渲染性能
   - 实现了对象池模式，优化资源管理

4. **数据驱动配置**
   - 实现了数据包支持，允许通过数据文件自定义字幕颜色
   - 创建了 `SubtitleTypeLoader` 类，支持加载和解析字幕类型配置
   - 添加了 `subtitle_types.json` 默认配置文件
   - 扩展了 `ColorCode` 类，支持从字符串名称转换为颜色枚举值
   - 修改了 `SubtitleEntryMixin`，使用数据加载器获取字幕颜色
   - 创建了数据包使用指南文档

5. **配置分享功能**
   - 实现了配置导出功能，将配置保存到 `shur_export.json` 文件
   - 实现了配置导入功能，从 `shur_import.json` 文件加载配置
   - 在配置界面添加了配置分享选项卡
   - 支持配置文件的序列化和反序列化
   - 添加了相关的本地化文本

6. **快捷键功能**
   - 实现了客户端快捷键系统，支持快速调整字幕设置
   - 切换字幕显示：快速开启/关闭 Minecraft 原生字幕
   - 增加/减小字幕缩放：快速调整字幕大小（范围 0.5-3.0）
   - 重新加载配置：从文件重新加载配置
   - 保存配置：保存当前配置到文件
   - 切换 iKun 彩蛋：开启/关闭 iKun 彩蛋
   - 增加/减小背景透明度：快速调整字幕背景透明度（每次约 10%）
   - 所有快捷键操作都支持操作反馈提示
   - 添加了完整的中英文本地化支持
