# Subtitle-Highlight-Unofficial-reset

## 原项目信息
- 原模组：Subtitle Highlight-https://www.mcmod.cn/class/8183.html

## 本项目说明
本项目为 **非官方重置版**，对 Minecraft 1.21.11 进行版本适配与兼容性修复，**不修改原模组核心功能**，后续将添加更多新功能和版本适配。
本项目完全继承原模组的 **GNU LGPLv3** 开源协议。

## 修改内容
- 适配 Minecraft 1.21.11 Fabric
- 保留原作者全部版权与功能逻辑
- 配置文件更名为 shur.json
- 新增命令系统：支持 `/shur` 命令进行配置管理
- 新增模组 API：提供 `SubtitleAPI` 供其他模组扩展功能
- 性能优化：视锥体检查、对象池模式
- 数据驱动配置：支持通过数据包自定义字幕颜色
- 配置分享功能：支持配置导出/导入
- 快捷键功能：支持快捷键快速调整字幕设置
